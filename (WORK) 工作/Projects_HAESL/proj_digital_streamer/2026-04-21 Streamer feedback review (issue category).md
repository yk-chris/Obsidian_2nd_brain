
| Category        | Desciption                                                                                                                                                                                                                                                                                                                                                                       |
| --------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Swap Module     | it is related to the off-module list to interact with the active streamer. In practice terms, the module can be selected from the off-module list to swap into the existing engine streamer on the Streamer Section as a planner role.                                                                                                                                           |
| Task Owner View | it is all about tasks allocated from the streamer and give the task owner to control their own tasks on the calendar view on task section. They can change the projected completion date (PCD), input actual completion date (ACD) with delayed reasons if any as well as request and approve the request if any trigger is needed.                                              |
| Notification    | On Notification, there are two area for notifying the task owners. Firstly, it is general message notfiication from the task actions like changing the task date from different planner or task owners. Another notification type is to notify the line manager as a supper task owner to accept or reject the approval requests from the staff under the same department group. |
| DIW/Cost Review | In specific streamer arrangement, the template with the label of DIW/Cost Review. Once the status of the active streamer is under the status of DIW/Cost Review, the planner would release the investigation or quotation after the first induction. Therefore, the planner is able to re-input the second induction information later on.                                       |
