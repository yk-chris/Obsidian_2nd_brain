## 🧠 核心洞察

### 1. AI 正在重構「計算光刻」整條產業鏈

光刻機的進步路線是三條腿並行：
- **λ（光源）**：EUV 已到 600W，目標 1kW，預計沿用至 2035–2040
- **NA（數值孔徑）**：High-NA EUV（0.55）預計 2026 下半年量產，主導未來 10 年；Hyper-NA（0.75+）估值逾 $600M，2030+ 才會到
- **K₁（工藝因子）**：**ILT（逆向光刻）× GPU 加速**是最大突破點——曲線光罩讓晶片提升相當於「三個製程代」，貢獻超過 High-NA

> 🔑 **投資含義**：ASML 的護城河不只是機器本身，而是整個「計算光刻軟體 + 量測 + 硬體」生態。NVIDA GPU 已成為 ASML ILT 計算的核心基礎設施——GPU 需求比想像中更多元。

**「3nm 謊言」**：所謂 3nm 晶片實際線寬遠超 20nm，芯片節點命名是行銷競爭的文字遊戲，真正衡量指標是電晶體密度。ASML 官方認為至少到 2040 年前，物理極限都不是瓶頸。

### 2. Tesla 的真實邏輯：飛輪 × 生態鎖定

Tesla Robotaxi 的擴張策略是刻意「低調堆積」：
```
更多無人里程 → 更好模型 → 更快城市擴張
→ 更多現金流 → 個人 FSD 更強 → 車輛需求與毛利更高
```

- Houston / Dallas：Model Y 已無人監督上路
- Las Vegas：Cybercab 真實街頭路測中
- 西班牙：50,000 miles 零事故，荷蘭媒體評語「無聊到完美」

**Cybertruck V2X**：PG&E 通過審核，加州史上首個 AC V2G 方案：
- 停電供電、尖峰賣電、最高 $4,500 補助
- 租賃方案：Premium AWD $997/月

> 🔑 **Jefferies 目標價 $350（Hold）**，Robotaxi + 人形機器人收入預計 **2027 年**才納入分析師模型——代表當前股價估值仍低估長期飛輪。Tesla 的生態護城河：你以為買了一輛車，實際上把家的能源未來全押在它的生態上。

### 3. 上市公司與華爾街短期主義的結構性張力

Duolingo 案例揭示了一個普遍困境：
- 上市前：DAU / 用戶體驗優先
- 上市後：EPS、ROE、短期 EPS 增長壓力主導決策

**長期主義者的三種路徑**：
1. 控制股（雙重股權結構）——Alphabet / Meta 模式
2. 拒絕上市——Cargill、IKEA
3. 選擇性上市——Berkshire 模式：找長期股東，不迎合短期市場

> 🔑 **估值啟示**：雙重股權結構公司（GOOG、META）在短期業績不佳時被市場懲罰的幅度往往被高估；創辦人控制 = 長期策略的「保險」。

### 5. Anthropic Opus 4.7 與 AI 工具競爭格局

**Claude Code 降智問題根源**：
- 長上下文中 Claude 3.7 推理不穩定 → 在代碼任務中忘記早期指令
- Opus 4.7 修復：混合推理架構（Fast Thinking + Deep Thinking 自動切換）

**Claude Design**：直接從 prompt → 生成可部署前端，威脅 Framer / Webflow 等無代碼設計工具。

**Mythos 提示詞框架**（Clip 0419）：舊提示詞為「指令性」設計，新模型（Opus 4+、GPT-5 等）需要「情境性」提示——給背景、目標、約束，不要給步驟。
