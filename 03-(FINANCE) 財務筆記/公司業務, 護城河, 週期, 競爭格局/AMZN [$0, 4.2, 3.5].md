## Step 2 護城河定性

| 護城河類型 | 判定 | 核心證據 |

| **品牌與網路效應** | **Y** | 全球最大電商平臺(第三方賣家>300萬)、2.3億Prime會員形成高粘性閉環；AWS行業領導地位(市佔率~32%)；廣告業務依附於電商流量壟斷[1][4] |
| **規模經濟** | **Y** | 物流網絡全球最密集(>400個配送中心)、單位物流成本業界最低；AWS邊際成本遞減；第三方平臺抽成模式可自動擴張[1] |
| **轉換成本** | **Y** | Prime會員年費+生態鎖定($139/年)；企業客戶遷移AWS成本極高(資料、應用架構重構)；廣告主依賴AMZN流量難以替代[1] |
| **技術與專利** | **Y** | AI/機器學習演算法(推薦、定價、需求預測)領先同業；AWS專有基礎設施；物流自動化專利積累(>8,000件與自動化相關)[1][5] |
| **成本優勢** | **Y** | 垂直整合(自營電商+物流+廣告)實現成本協同；AWS規模優勢使單位計算成本最低；供應鏈效率全球領先[4] |
| **政策與監管** | **N** | 面臨全球反壟斷調查(歐盟、美國)；英國與歐盟已啟動正式調查；物流與勞工政策風險上升 |

## Step 3 週期性定位

### 週期定位基本參數

| 參數 | 數值 | 說明 |
|-----|-----|------|
| **週期來源** | 企業IT支出週期 + 消費品景氣週期 + 廣告主預算週期 | AWS收入驅動因素為企業資本支出(3-5年週期)；零售受消費景氣影響；廣告隨經濟擴張收縮 |
| **典型週期長度** | 3-5年 | 一個完整企業IT投資週期；消費景氣週期通常3-7年 |
| **週期振幅(歷史EBIT波幅%)** | 35-45% | EBIT從谷底($15B)至高峰($65B+)，振幅約330%；毛利率波動20-35%反映產能利用率變化[1] |
| **風險承擔者** | AWS與廣告(總利潤貢獻60%)；零售業務充當週期避險資產 | 高利潤業務敏感度高；低利潤零售提供穩定現金流 |
| **目前週期位置** | **上升中段(景氣復甦期)** | AI投資推動AWS高成長(預期30%+YoY增速)；廣告主支出恢復；消費品需求回溫[1][5] |

### 週期位置四指標速查表

| 指標 | 現狀 | 解讀 |
|-----|-----|------|
| **庫存水位** | 正常-偏低 | 雲端資源按需配置，無傳統庫存堆積；電商庫存周轉加快(AI需求預測準度提升) |
| **Capex動向** | **激增**(YoY+40-60%) | AI/機器學習基礎設施投資爆發；AWS資料中心擴張全球加速；週期位置**明確上行信號**[1][5] |
| **訂單能見度** | 高(AWS長期合約比例>50%) | 企業客戶簽署3年+承諾合約，說明需求確定性強 |
| **現貨vs合約價格** | 現貨價格上升 | AWS計算資源現貨成交價格上漲15-20%，反映緊張供應與高需求[1] |

**綜合判斷：處於週期上升階段中段，Capex激增與訂單能見度高為核心確認信號。**

## Step 5 快速小結

① **一句話業務定義**：垂直整合的全球數位商務與雲端基礎設施帝國，通過電商(零售+第三方市場)與AWS提供規模化解決方案，廣告與訂閱為新利潤引擎。[1][4]

② **護城河標籤**：**網路效應+規模經濟+轉換成本+技術領先 / 4.2 / 5**｜最大威脅：反壟斷監管瓦解市場壟斷地位。[1][4][5]

③ **週期性標籤**：**企業IT支出驅動 / 3-5年 / 上升中段**｜目前處景氣復甦期，Capex激增與AWS訂單能見度高為核心確認信號。[1][5]

④ **矩陣象限定位**：座標 [4.2, 3.5]，**第①象限**｜在護城河深厚與週期上升的交集點，但監管風險為未來主要變數。

⑤ **與同類公司最大結構性差異**：AMZN唯一擁有**"零售現金奶牛+高利潤雲端引擎"雙驅動模式**；MSFT依賴訂閱穩定性更強但成長天花板明確；GOOGL與Meta廣告依賴度過高景氣脆弱性最大；IBM雲端轉型已落後無法追趕。[1][4]
