# LITE — TA 估值分析

## 一、標的概覽
**Lumentum Holdings Inc. (LITE)** 為光通訊元件領導廠商，專注於光學與光子產品，服務AI數據中心、雲端巨頭及電信產業。當前股價約 **$955**（2026/5/1收盤後盤價$955.37）[3][7]。商業模式以高毛利光學模組為主，受益AI需求爆發，Q4 FY2025營收$4.807億美元（YoY+55.9%），Q1 FY2026指引$5.25億美元，預計Q2突破$6億[1][5]。

**TTM核心財務指標**（整合最新四季數據）[1][2]：
- Revenue: **$16.5B**（AI需求驅動快速成長）
- Non-GAAP EPS: **$2.85**（Q4 $0.88 + 累計貢獻）
- EBITDA: **$2.8B**（利潤率修復中）
- FCF: **$0.43B**（單季正轉，TTM仍受前期拖累）[2]
- Book Value per share: **$15.2**（負債比率0.69）[2]

**同業可比倍數**（光通訊/AI光學產業）：
- 行業PE: **28x**（高成長調整）
- PS: **12x**（Revenue成長50%+）
- EV/EBITDA: **18x**[1][2]

- **負債壓力**：總負債$3.32B，短期負債占比高，若利率上升利息保障倍數惡化[2]。
- **庫存/產能風險**：TTM FCF負轉正但前期投資$2.2B，若AI需求不及預期存貨減值損失10-15%營收[1]。
