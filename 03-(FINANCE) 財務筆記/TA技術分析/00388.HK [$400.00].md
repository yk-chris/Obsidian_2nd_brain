## 三、估值模型詳細計算

### 3.1 DCF 現金流折現法

**模型假設：**
- 預測期：5 年（2026–2030）
- 年均 FCF 增長率：12%
- WACC（加權平均資本成本）：8.5%
- 永續增長率（終值）：3%
- TTM FCF 基數：38.00 億港元

**逐年現金流折現：**

| 年度 | FCF（億港元） | 折現因子 | 現值（億港元） |
|-----|------------|--------|------------|
| 2026 | 42.56 | 0.9217 | 39.23 |
| 2027 | 47.66 | 0.8495 | 40.47 |
| 2028 | 53.38 | 0.7829 | 41.78 |
| 2029 | 59.78 | 0.7218 | 43.13 |
| 2030 | 66.96 | 0.6650 | 44.53 |
| **PV(FCF 2026-2030)** | — | — | **209.14** |

**終值計算：**
- 2031 年 FCF（永續基數）= 66.96 × 1.03 = 69.00 億港元
- 終值 = 69.00 / (8.5% - 3%) = 69.00 / 5.5% = 1,254.55 億港元
- PV(終值) = 1,254.55 / 0.6650 = 834.38 億港元

**企業價值及股權價值：**
- 企業價值 = 209.14 + 834.38 = 1,043.52 億港元
- 減：淨債務 = 0（交易所通常現金充足）
- 股權價值 = 1,043.52 億港元
- 流通股本 = 4.8 億股
- **每股公允價值 = 1,043.52 / 4.8 = $420.00**
- **偏離幅度 = (420.00 - 414.40) / 414.40 = +1.4%（略低估）**

### 3.3 EV/EBITDA 企業價值倍數法

**行業倍數參考：**
- 成熟交易所 EV/EBITDA：20–28 倍
- 高增長交易所 EV/EBITDA：18–25 倍
- **採用保守倍數：25 倍**

**計算過程：**
- TTM EBITDA = 70.00 億港元
- EV/EBITDA 倍數 = 25 倍
- 企業價值 = 25 × 70.00 = 1,750.00 億港元
- 減：淨債務 = 0（假設無淨債務）

**反向驗證現價隱含倍數：**

- 隱含 EV/EBITDA = 1,987.00 / 70.00 = **28.4 倍**
