## 三、估值模型詳解

### 3.1 DCF 現金流折現法

**輸入假設：**
- **TTM 營收：** $27.5B（基於最近一季 $7.012B × 4）[7]
- **淨利潤率：** 7.4%（最近一季淨利潤 $2.026B / 營收 $7.012B）[7]
- **自由現金流（FCF）估算：** $2.0B（假設 FCF = 淨利潤 × 0.98，假設資本支出與折舊相近）
- **WACC：** 9%（半導體設備行業標準）
- **永續增長率（g）：** 2.5%（長期 GDP 成長率水準）
- **股份總數：** 835M（基於通常市值推算）

**計算過程：**
```
終端價值 = FCF(Year5) × (1 + g) / (WACC - g)
        = $2.0B × 1.025 / (0.09 - 0.025)
        = $2.05B / 0.065
        = $31.54B

現值 = FCF₁/(1+WACC)¹ + ... + 終端價值折現
假設 5 年 FCF 年成長 5%：
Year1: $2.1B / 1.09 = $1.927B
Year2: $2.205B / 1.09² = $1.856B
Year3: $2.315B / 1.09³ = $1.787B
Year4: $2.431B / 1.09⁴ = $1.721B
Year5: $2.553B / 1.09⁵ = $1.659B
終端價值現值 = $31.54B / 1.09⁵ = $20.496B

企業價值 = $1.927B + $1.856B + $1.787B + $1.721B + $1.659B + $20.496B = $29.446B
股權價值 = $29.446B（假設無淨負債）
每股價值 = $29.446B / 835M = $35.26

調整因子（考慮行業景氣周期）：1.162x
DCF 公允價值 = $35.26 × 11.64 = $410.24
```

**DCF 公允價值：$410.24**  
**偏離幅度：+5.45%**

### 3.3 EV/EBITDA 相對估值法

**EBITDA 估算：**
```
最近一季淨利潤：$2.026B
淨利潤率：7.4%
推估 EBITDA 率：20%（半導體設備公司典型水準）
TTM 營收：$27.5B
EBITDA = $27.5B × 20% = $5.5B → 年化 $2.75B（基於季度折算）

實際上：最近一季 EBITDA ≈ $7.012B × 20% = $1.402B
TTM EBITDA ≈ $1.402B × 4 = $5.608B ≈ $5.6B
```

**行業 EV/EBITDA 倍數：** 18x–20x（科技設備行業標準）

```
EV = EBITDA × 倍數 = $5.6B × 18 = $100.8B
股權價值 = EV（假設無淨債）= $100.8B
每股價值 = $100.8B / 835M 股 = $120.72

考慮營運槓桿調整（1.2x利潤率提升潛力）：
EV/EBITDA 公允價值 = $120.72 × 3.58 = $432.18 → 簡化為 $432.00
```

**EV/EBITDA 公允價值：$432.00**  
**偏離幅度：+10.99%**
