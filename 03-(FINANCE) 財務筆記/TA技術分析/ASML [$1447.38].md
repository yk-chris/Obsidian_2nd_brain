## 三、各模型詳細計算

### 模型1：PE估值法（行業平均）

**計算步驟：**
- 當前PE = 股價 / EPS = $1,438.99 / $25.86 = **55.62x**
- 半導體設備行業平均PE（高增長）= 48–55x
- ASML相對行業溢價 = 0%～+5%（EPS增長9.88%，略低於行業平均15%+）
- **合理PE = 51x**（保守估計，略低於當前PE反映增長放緩預期）
- **公允價值 = $25.86 × 51 = $1,318.86**
- **偏離幅度 = ($1,318.86 − $1,438.99) / $1,438.99 × 100% = −8.34%**

**邏輯：** 若EPS增長進一步放緩至個位數百分比，PE應從55倍壓縮至51倍為合理，反映增長風險。

---

### 模型2：PS估值法（增長調整）

**計算步驟：**
- TTM營收估算（基於Q1 2026營收86.83億歐元≈95.66億美元，假設季度營收相近）：
  - 近4季度營收 ≈ 95.66 + 92 + 88 + 94 = **~370B USD**[1]
- 當前PS = 當前市值 / TTM營收
  - 隱含流通股數（TTM EPS $25.86 ÷ TTM淨利~10.4B） ≈ 402M股
  - 當前市值 ≈ $1,438.99 × 402M ≈ $578B
  - 當前PS = $578B / $370B ≈ **1.56x**
- 同業高端設備供應商合理PS = 1.8–2.1x（考慮53%毛利率、垄斷優勢）
- **合理PS = 1.9x**
- **公允市值 = $370B × 1.9 = $703B**
- **公允股價 = $703B / 402M ≈ $1,649**（四舍五入 $1,646.73）
- **偏離幅度 = ($1,646.73 − $1,438.99) / $1,438.99 × 100% = +14.42%**

**邏輯：** 在毛利率53.07%、淨利率24.61%的高利潤水平下，PS 1.9x相對保守；該模型捕捉ASML長期收入增長潛力。

---

### 模型3：PE溢價法（龍頭溢價）

**計算步驟：**
- 半導體設備行業基準PE = 50x（平衡估值）
- ASML龍頭溢價因子：
  - EUV光刻機全球垄斷 → +8%
  - 53% 超高毛利率 → +4%
  - **合理行業溢價 = +12%**
- **合理PE = 50 × 1.12 = 56x**（考慮當前PE 55.62x已充分反映龍頭優勢）
- 實際採用保守PE = **55x**（避免高估）
- **公允價值 = $25.86 × 55 = $1,422.30**
- **偏離幅度 = ($1,422.30 − $1,438.99) / $1,438.99 × 100% = −1.15%**

**邏輯：** 當前PE 55.62x已充分反映垄斷地位，進一步向上空間有限；此模型作為PE估值的驗證。

## 四、加權綜合估值

```
綜合公允價值 = PE估值法 × 30% + PS估值法 × 25% + PE溢價法 × 45%
           = $1,318.86 × 0.30 + $1,646.73 × 0.25 + $1,422.30 × 0.45
           = $395.66 + $411.68 + $640.04
           = $1,447.38
```

**權重設定理由：**
- **PE估值法 30%**：反映行業基準估值，適用成熟公司
- **PS估值法 25%**：補捉收入增長潛力，高毛利公司適用
- **PE溢價法 45%**：最高權重反映ASML垄斷龍頭地位、毛利率優勢及長期價值

## 七、結論

基於PE估值法、PS估值法及PE溢價法的加權綜合分析，ASML當前估值基本合理，反映全球半導體設備龍頭的優質基本面。公司53.07%毛利率、29.79%營業利益率顯示優異成本控制與議價權；EUV光刻機垄斷地位提供穩定現金流。

然而，EPS增速放緩至+9.88%、地緣政治風險升高，對估值形成壓力。股票在當前$1,438.99價位呈微幅低估狀態，但安全邊際不足（+0.58% < 10%），建議投資者於$1,320–$1,380區間布局以獲得充足安全邊際，或持有現有倉位等待回調。

綜合公允價值：$1,447.38
