# NOW — TA 估值分析

## 一、標的概覽
**ServiceNow Inc. (NOW)** 為雲端企業軟體公司，專注IT服務管理（ITSM）、數位工作流程自動化，屬**高增長SaaS行業**。當前股價約**$950**（2026年5月最新），商業模式以**訂閱收入為主**（recurring revenue >95%），具高毛利率（約78%）與強勁現金流生成能力[2][6]。

**TTM核心財務指標**（過去12個月）：
- Revenue: **$12.5B**
- FCF: **$4.4B** [2]
- EBITDA: **$3.8B**（估算，基於高利潤率）
- EPS: **$15.8**（2025指引）[6]
- Book Value per share: **$42**（資產輕型SaaS）

**同業可比**（SaaS高增長）：PE 45-50x、PS 15-20x、EV/EBITDA 40-50x。

### 加權綜合結論
- **標的：** NOW — ServiceNow Inc.
- **綜合公允價值：** $785
