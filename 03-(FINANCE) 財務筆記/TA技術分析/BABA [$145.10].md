### 各模型詳細計算

#### 1. DCF 現金流折現法

**假設條件：**
- FCF TTM：$10.1B
- 未來 5 年增長率：4%（考慮利潤波動，保守估計）
- WACC：8%（科技平台企業，中等風險）
- 永續增長率：3%（成熟經濟體企業）

**計算步驟：**
- 預測期 FCF：$10.5B → $10.9B → $11.3B → $11.8B → $12.3B
- 預測期 PV（折現因子 8%）：$43.8B
- 終值 = FCF₅ × (1+g) / (WACC-g) = $12.3B × 1.03 / 0.05 = $253.0B
- 終值現值（第 5 年折現）：$172.1B
- 企業價值 = $43.8B + $172.1B = $215.9B
- **每股公允價值 = $215.9B ÷ 2.34B 股 = $152.00**
- 偏離幅度 = (+16.5%)

#### 2. PE 相對估值法

**假設條件：**
- 正常化 EPS：取最近三期平均 = ($7.04 + $9.04 + $12.52) ÷ 3 = **$9.53 USD**
- 行業平均 PE 倍數：20 倍（中文互聯網平台龍頭）
- 考慮利潤波動風險，採用保守的正常化 EPS = $9.0

**計算：**
- 公允價值 = $9.0 × 20 = **$180.00**
- 偏離幅度 = (+38.1%)

#### 3. PS 相對估值法（相對估值輔助）

**假設條件：**
- TTM 營收：284.84B（年度級別）[1]
- 行業平均 PS 倍數：1.4 倍（中文電商龍頭）
- 市場平均 PS：1.2-1.6 倍

**計算：**
- 市場隱含營收價值 = $305.21B ÷ $284.84B = 1.07 倍（當前）
- 公允價值市值 = $284.84B × 1.4 = $399.8B
- **每股公允價值 = $399.8B ÷ 2.34B 股 = $170.77（調整）**
- 取保守值 **$98.00**（假設 PS 回落至 1.0 倍的悲觀情景）
- 偏離幅度 = (-24.9%)

#### 4. EV/EBITDA 法

**假設條件：**
- 推估年度 EBITDA ≈ 營收 × 18% = $284.84B × 0.18 ≈ $50B
- 企業現值 EV ≈ $305.21B（市值）+ 淨債務（低）
- 行業 EV/EBITDA 倍數：5.0 倍（穩健成長階段）

**計算：**
- 公允市值 EV = $50B × 5.0 = $250B
- **每股公允價值 = $250B ÷ 2.34B 股 = $107.00**
- 偏離幅度 = (-18.0%)
