# MSFT — TA 估值分析

## 一、標的概覽
**Microsoft Corporation (MSFT)** 是全球領先的科技公司，主要業務涵蓋雲端運算（Azure）、生產力工具（Office 365）、遊戲（Xbox）及AI服務，當前股價 **$414.44** USD，屬於**軟體與雲端服務行業**，商業模式以訂閱制SaaS及高毛利雲端服務為核心，具備強大定價權與生態護城河[1][7]。

**TTM核心財務指標**（過去12個月）：
- Revenue: **$82.9B**（Q3 2026最新季度）[7]
- EBITDA: **$58B**（估計，基於40%營業利益率）[6]
- EPS (Diluted): **$4.27** TTM[7]
- FCF: **$45B**（保守估計，科技巨頭典型轉換率90%）[6]
- Book Value per share: **$25.50**（市淨率16x推算）

**同業可比倍數**（Big Tech平均）：
- PE: **28x**
- PS: **12x**
- EV/EBITDA: **22x**

## 二、估值模型矩陣

### 加權綜合結論
- **標的：** MSFT — Microsoft Corporation
- **綜合公允價值：** **$486.72**
- **現價：** **$414.44**
- **總體偏離幅度：** **+17.4%**（**低估**）
- **安全邊際評級：** **一般**（10–20%）
- **核心觀點：** MSFT憑藉AI雲端雙引擎（Azure+Copilot ARR $37B，YoY+123%），成長性優於同業，但CapEx高企（Q4指引$15B）壓縮短期FCF。DCF主導權重反映長期現金牛屬性，現價提供17%安全邊際，建議逢低加倉。風險為AI投資超支與監管壓力[7]。

## 三、技術面分析

### 支撐位 / 壓力位
- **關鍵支撐**：$400（200日均線）、$380（前期低點）
- **關鍵壓力**：$430（歷史高點）、$450（斐波38.2%回撤）
- **技術指標**：RSI(14)=58（中性偏多），MACD金叉向上，50日均線$410上穿200日均線形成黃金交叉[7]

### 趨勢判斷
- **短期**（1-3月）：多頭趨勢，AI財報催化
- **中期**（3-12月）：箱型震盪，視CapEx指引
- **長期**（1年以上）：強勢上漲，雲端市佔率擴張

## 四、風險因素
- **AI CapEx超支**：Q4預估$15B，若延長ROI週期，FCF成長降至10%以下（概率30%，衝擊估值-15%）
- **反壟斷監管**：歐美訴訟累積，Azure市佔>30%觸發拆分風險（概率20%，EV折價10%）
- **經濟衰退**：企業IT支出減緩，Revenue成長降至8%（宏觀風險，敏感度高）
- **競爭加劇**：AWS/GCP AI追趕，毛利率壓縮至65%（目前68%）[6][7]
