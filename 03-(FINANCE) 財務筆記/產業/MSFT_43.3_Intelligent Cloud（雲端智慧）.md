# MSFT — Intelligent Cloud（雲端智慧）

## 業務部門概覽

Intelligent Cloud 為 Microsoft 最大收入部門，涵蓋 Azure 公有雲（IaaS/PaaS/SaaS）、Azure AI Services（含 Azure OpenAI Service）、SQL Server、Windows Server、GitHub、Nuance 及企業支援服務。Azure 為全球第二大公有雲平台，負責承載企業工作負載遷移、資料分析、AI 推理/訓練及混合雲（Azure Arc）等核心需求，是 Microsoft 成長最快及估值最高的業務引擎。

## 收入與成本結構

- **收入貢獻（Revenue %）**：43.3%（FY2024 全年收入約 $106.2B）
- **毛利率**：72.1%（FY2024）
- **主要成本驅動因素**：資料中心 CapEx（伺服器、GPU、冷卻）、研發及工程人力、AI 模型授權（OpenAI 合作）、網絡帶寬、電力能源

## 投資回報分析（invest-cost-return）

- **投入資本（Invested Capital）**：CapEx 大幅擴張中，FY2024 全公司 CapEx ~$55.7B，Intelligent Cloud 佔大部分
- **ROIC / 部門資本回報率**：部門層級未單獨披露；整體 MSFT ROIC ~30%+
- **FCF 貢獻**：高邊際利潤業務，Azure 雲端服務毛利率 >70%，FCF 貢獻顯著

## 生產要素成本分析

### Intelligent Cloud（雲端智慧）

**市場概況**

雲端基礎設施（IaaS/PaaS）全球市場規模約 $6,800億（2024），預計至2030年以 ~18% CAGR 擴張。Azure 為全球第二大公有雲平台，市佔率約 23%，僅次於 AWS（32%），遠超 Google Cloud（12%）。競爭格局屬**寡頭壟斷**，三家主要供應商合計佔全球市場逾 67%。Microsoft 的差異化優勢在於：企業端深度整合（Active Directory、Office 365、Teams）、混合雲解決方案（Azure Arc）、以及與 OpenAI 的獨家戰略合作將 AI 能力（Azure OpenAI Service、Copilot Studio）無縫嵌入雲端生態，形成強大的企業用戶黏性與切換成本護城河。
