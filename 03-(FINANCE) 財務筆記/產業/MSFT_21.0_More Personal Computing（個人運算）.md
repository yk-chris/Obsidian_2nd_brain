# MSFT — More Personal Computing（個人運算）

## 業務部門概覽

此部門涵蓋 Windows OEM（授權）、Xbox（主機硬件 + Game Pass 訂閱 + Activision-Blizzard 遊戲 IP）、Surface 硬件、Bing 搜索廣告及 Microsoft Store。Activision-Blizzard 收購（$68.7B，2023年完成）將 Call of Duty、Candy Crush、Diablo、Overwatch 等頭部 IP 納入 Xbox 生態，大幅提升遊戲內容資產。此部門整體增長放緩，但 Xbox Game Pass 訂閱化及 Bing Copilot 廣告是結構性成長引擎。

## 收入與成本結構

- **收入貢獻（Revenue %）**：21.0%（FY2024 全年收入約 $46.3B）
- **毛利率**：49.3%（FY2024，受硬件低毛利拖累）
- **主要成本驅動因素**：遊戲內容開發成本（Activision-Blizzard + Xbox Studios）、硬件製造成本（Surface/Xbox）、銷售行銷、Windows OEM 維護、廣告技術基礎設施

## 投資回報分析（invest-cost-return）

- **投入資本（Invested Capital）**：Activision 收購商譽 ~$69B 為主要資產；Xbox 主機硬件虧損銷售（刀片模式）
- **ROIC / 部門資本回報率**：硬件業務 ROIC 偏低；遊戲訂閱（Game Pass）及軟件（Windows）邊際回報高
- **FCF 貢獻**：低於其他兩個部門，受硬件成本及 Activision 攤銷影響

## 生產要素成本分析

### More Personal Computing（個人運算）

**市場概況**

全球 PC 市場出貨量 2024 年約 2.55億台（IDC），以 ~2–3% 低增速復甦；PC OEM 市場競爭格局為**高度集中寡頭**（Lenovo、HP、Dell 合計 ~60% 出貨），Microsoft 收取 Windows 授權費，商業模式為後端分潤。Xbox Game Pass 面對 PlayStation Plus（Sony）及 Nintendo Switch Online，市場格局為三方鼎立。Bing 廣告受 Copilot 整合帶動提升搜索份額（目前 ~4–5%），但仍遠落後於 Google。Activision-Blizzard 收購（$68.7B，2023年完成）將 Xbox 遊戲部門收入大幅提升，Call of Duty、Candy Crush、Diablo 成為新增收入引擎。
