# ASML — Deep UV 系統

## 業務部門概覽

Deep UV 系統是 ASML 非 EUV 微影設備的核心產品線，主要應用於記憶體晶片製造領域。該業務透過提供深紫外線光刻技術，協助全球半導體廠商進行 DRAM 和 NAND 快閃記憶體的製程工藝開發。Deep UV 系統在公司整體淨系統銷售額中佔有重要地位，是傳統微影技術領域的旗艦產品。

## 生產要素成本分析

### Deep UV 系統

**市場概況**

Deep UV 微影設備市場為成熟的寡頭競爭格局，主要應用於 DRAM 及 NAND 快閃記憶體製造。全球市場規模受記憶體晶片需求週期驅動，2024–2025 年因生成式 AI 應用推升高效能運算晶片需求，相關市場需求持續復甦[2]。ASML 在微影設備領域市場佔有率超過 80%，具有絕對領導地位[7]，客戶包括全球十大半導體廠商[7]。競爭對手主要為日本的尼康（Nikon）及佳能（Canon），但技術差距明顯。ASML 的差異化優勢在於先進光學設計、製程整合能力與客戶支持體系；然而，2026 年 Deep UV 業務預期因中國客戶動態調整而面臨下行壓力[1]。
