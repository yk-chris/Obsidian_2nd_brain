# BABA — 雲智能集團

## 業務部門概覽

> 阿里雲智能集團（Alibaba Cloud Intelligence Group）是中國最大雲計算服務商，亦是全球第三大雲平台（僅次於 AWS 及 Azure），提供 IaaS（算力基礎設施）、PaaS（平台服務）及 SaaS（軟件服務）。近期以 AI 大模型（通義千問）及 AI-Native 雲服務為核心增長引擎，FY2025 Q4 收入增速加速至 18%，AI 相關產品收入實現三位數增長。目標客戶為中國大型企業、政府及國際市場。

## 收入與成本結構

- **收入貢獻（Revenue %）**：12.0%（FY2025，約 USD 165億）
- **毛利率**：~55.0%（公有雲及 AI 服務混合，趨勢向上）
- **主要成本驅動因素**：數據中心建設及電力成本、AI 晶片（GPU）採購、研發人員薪酬、網絡帶寬

## 投資回報分析（invest-cost-return）

- **投入資本（Invested Capital）**：極高（阿里巴巴宣佈未來5年投入超過5,000億人民幣於 AI 及雲基礎設施）
- **ROIC / 部門資本回報率**：中等（公有雲盈利能力提升，但 AI 基礎設施大規模投入壓縮短期回報）
- **FCF 貢獻**：正轉中（由虧損轉盈利，AI 服務拉動高毛利收入結構改善）

## 生產要素成本分析

### 雲智能集團

**市場概況**

中國雲計算市場2024年規模約人民幣6,600億元，預計2028年突破1.5兆元，增速約20%。阿里雲市佔率約36%穩居第一，其次為華為雲（19%）、騰訊雲（15%）及天翼雲（10%）。全球範圍內，AWS、Azure 及 Google Cloud 主導市場，阿里雲主要佈局亞太及中東市場。AI 大模型浪潮帶動 GPU 算力需求爆發，阿里雲的通義千問系列大模型已成為企業 AI 轉型首選之一，並以 API 調用方式快速滲透中小企業客戶，形成 API 調用量三位數增長的新增長飛輪。
