# TSM — 高效能運算平台（HPC）

## 業務部門概覽

高效能運算平台（HPC）涵蓋**人工智慧與5G應用**場景，是台積電當前最大的營收來源與主要成長引擎。該平台主要為NVIDIA、AMD、Apple等一流客戶提供AI數據中心晶片、邊緣AI運算及5G相關應用的先進製程代工服務，核心客戶包括輝達用於訓練AI演算法的高階數據中心晶片。

## 第二部分：市場概況

**市場規模與增長勢頭**

台積電整體營收預計2026年增速突破30%[1]，其中HPC平台增長最為迅速，佔比從2025年Q4的55%快速提升至2026年Q1的61%，呈現加速集中趨勢[1][5]。管理層預測到2030年，每年數據中心資本支出規模可能達3至4兆美元，為HPC業務奠定長期成長基礎[2]。

**主要競爭對手與競爭地位**

台積電在**全球AI晶片製造領域處於壟斷性領導地位**，客戶群包括NVIDIA（AI訓練晶片）、AMD、Apple等業界頂尖企業[2]。這些客戶高度依賴台積電的5奈米、3奈米及2奈米等尖端製程技術[1]。先進製程佔晶圓銷售總額74%，其中5奈米佔36%、3奈米佔25%、2奈米已進入量產，技術領先幅度難以被短期追趕[1]。

## 第四部分：投資回報分析

- **投入資本（Invested Capital）**：
  台積電2026年資本支出提升至520億至560億美元[1]，資金集中投向3奈米與2奈米產能擴張，直接支撐HPC平台客戶的長期訂單需求。

- **ROIC / 部門資本回報率**：
  搜索結果未公開該部門具體ROIC。然企業層級毛利率及營業利益率已顯著提升[1]，HPC平台作為高利潤率業務主體，資本回報率預計處於高位。

- **FCF貢獻**：
  HPC平台佔總營收61%[1]，為自由現金流主要來源。營收增速超30%[1]、毛利率提升驅動下，該平台對集團FCF貢獻佔比預計不低於50%。

- **成長驅動因素**：
  **AI運算需求爆炸性增長**是核心引擎，特別是NVIDIA等客戶對AI訓練晶片的強勁需求[2]。先進製程持續演進（5nm、3nm、2nm梯次導入）鞏固性能領先[1]。邊緣AI運算與5G應用延伸提供增長新動力[1]。

- **競爭地位**：
  台積電在HPC高效能晶片領域保持**不可撼動的壟斷性地位**[2]。一流客戶群粘性高、長期合作協議提供業務穩定性。地緣政治與成本上漲構成短期風險因素，但尚未撼動其市場龍頭地位[1]。
