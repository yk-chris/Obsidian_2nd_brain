# BABA — 國際數字商業集團（AIDC）

## 業務部門概覽

> 阿里巴巴國際數字商業集團（AIDC）涵蓋面向海外消費者的 AliExpress（速賣通）、Trendyol（土耳其最大電商，阿里持股約69%）、Lazada（東南亞電商）及面向企業客戶的 Alibaba.com（B2B 跨境貿易平台）。FY2025 收入增速達29%，為集團內增速最快業務。目標市場為歐洲、中東、東南亞及土耳其等新興及成熟跨境電商市場。

## 收入與成本結構

- **收入貢獻（Revenue %）**：10.0%（FY2025，約 USD 137億）
- **毛利率**：~32.0%（跨境電商含物流成本，毛利率低於國內電商）
- **主要成本驅動因素**：跨境物流（菜鳥）費用、用戶補貼（Choice 免費配送）、行銷推廣費、平台建設及當地人員成本

## 投資回報分析（invest-cost-return）

- **投入資本（Invested Capital）**：高（持續補貼 AliExpress Choice 以搶奪市場、擴建海外倉）
- **ROIC / 部門資本回報率**：低（整體仍虧損，戰略性市場擴張期）
- **FCF 貢獻**：負（重投入擴張期，依賴核心電商及雲業務現金流）

## 生產要素成本分析

### 國際數字商業集團（AIDC）

**市場概況**

全球跨境電商市場規模預計2025年超過USD 1兆，年增速約20%。阿里巴巴以 AliExpress 主攻歐洲（特別是西班牙、法國、波蘭）及中東市場，以 Trendyol 深耕土耳其（市場份額超60%），以 Lazada 佈局東南亞（但面臨 Shopee 強力競爭壓力）。主要競爭對手包括 Temu（拼多多旗下，近年在歐美強勢擴張）、Shein（快時尚跨境電商）及 Amazon（歐美本土強者）。Trendyol 為集團內盈利最強的 AIDC 資產，有望率先獨立 IPO。

**生產要素成本分析**

| 生產要素 | 成本佔比 | 要素來源 | 在生產中的作用 | 重要性評分（1–5） |
| -------- | -------- | -------- | -------------- | ----------------- |
| 跨境物流（運費及倉儲） | 38.0% | 菜鳥網絡（關聯方）+ 第三方快遞 | 實現跨境商品履約，直接影響用戶體驗及留存 | 5／5（跨境電商最核心成本，配送速度決定競爭力） |
| 用戶補貼（Choice 免費配送等） | 25.0% | 內部預算 | 吸引新用戶，對抗 Temu/Shein 價格競爭 | 5／5（高競爭環境下市場份額爭奪的必要投入） |
| 行銷及廣告（海外市場） | 18.0% | Google/Meta/TikTok 等廣告平台 | 提升品牌知名度及獲取新用戶 | 4／5（規模效應可降低單位獲客成本） |
| 本地化運營人員 | 12.0% | 各目標市場本地招募 | 商戶管理、客服及合規支持 | 3／5（必要成本，但可部分自動化） |
| 技術研發（本地化平台） | 7.0% | 總部技術輸出 + 本地工程師 | 適配各市場語言、支付及物流生態 | 4／5（本地化深度影響市場滲透率） |
