# GOOGL — Google Cloud

## 業務部門概覽
Google Cloud 提供雲端基礎設施（IaaS）、平台服務（PaaS）及軟體即服務（SaaS），包括 GCP、Google Workspace、Anthos 等。目標市場為企業客戶，涵蓋 AI/ML、資料分析、大數據及辦公協作工具。（68字）[3]

## 第二部分：市場概況

全球雲端運算市場 2024 年規模約 6,250 億美元，預計 2025-2030 CAGR 16.5% 成長至 2.3 兆美元。競爭格局為寡頭市場，前三大（AWS 31%、Azure 25%、Google Cloud 11%）佔比 67%。主要競爭對手為 Amazon AWS、Microsoft Azure。Google Cloud 以 AI/ML 領導地位（TPU 晶片、Gemini 模型）及資料分析優勢（BigQuery）差異化，2024 年成長最快，企業採用率快速提升，特別在生成式 AI 應用領域。（152字）[3][4]

- **投入資本（Invested Capital）**：約 800 億美元，重資產模式。
