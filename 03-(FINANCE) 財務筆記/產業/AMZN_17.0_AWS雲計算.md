# AMZN — AWS雲計算

## 業務部門概覽
AWS是亞馬遜雲科技業務，提供IaaS、PaaS、SaaS服務，包括EC2、S3、Lambda等，服務全球企業AI、大數據、應用託管需求。2024年收入約1287億美元，貢獻公司58%經營利潤，是最高毛利業務。[1][2][4]

## 第二部分：市場概況

全球雲計算市場規模約8000億美元，2025-2030年CAGR預計25%，AI工作負載驅動爆發式增長。主要競爭對手為Microsoft Azure、Google Cloud，市場格局寡頭競爭（前三家市佔超65%）。AWS市佔率31%，領先優勢來自最早進入市場（2006年）、最完整服務生態（200+產品）及混合雲解決方案，客戶黏性高，遷移成本壁壘強大。[2][4][5]
