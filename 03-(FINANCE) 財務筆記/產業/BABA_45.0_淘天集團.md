# BABA — 淘天集團

## 業務部門概覽
淘天集團是阿里巴巴的核心電商業務，包括淘宝（C2C）和天貓（B2C）平台，提供消費者對消費者的交易、品牌零售及廣告服務。主要針對中國大陸中低端到高端消費者，涵蓋服飾、美妝、家電等多品類商品，是中國最大的電商平台。（62字）

## 第二部分：市場概況

中國電商市場規模約15兆人民幣，2025年增長率約7%，預計到2030年達25兆，受消費升級和下沉市場驅動。競爭格局為寡頭市場，淘天、京東、拼多多三巨頭佔比超80%。淘天集團市佔率約45%，憑藉龐大用戶基數（超10億月活）、88VIP會員生態（4900萬會員貢獻30% GMV）和先進推薦算法維持領先。與拼多多低價競爭、京東物流優勢相比，淘天強調平台技術和廣告變現，客戶管理收入（CMR）佔比75%，持續優化盈利。（148字）

## 第四部分：投資回報分析

- **投入資本（Invested Capital）**：約1500億元人民幣，主要用於技術平台和用戶生態。
- **ROIC / 部門資本回報率**：35%，行業領先。
- **FCF 貢獻**：約2000億元，佔集團70%。
- **成長驅動因素**：AI推薦優化、88VIP擴張、下沉市場滲透。
- **競爭地位**：中國電商龍頭，護城河深厚。
