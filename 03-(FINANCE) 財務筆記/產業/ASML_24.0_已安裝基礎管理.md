# ASML — 已安裝基礎管理

## 業務部門概覽
已安裝基礎管理（Installed Base Management, IBM）部門主要提供售後服務、系統升級、軟體維護及計量檢測解決方案，針對全球半導體製造商已安裝的ASML光刻設備進行支援。核心服務包括EUV系統生產力升級（如NXE:3800E）、定期維護及高階服務，目標市場為台積電、Intel、三星等先進製程晶圓廠，佔公司總收入約24-25%，為穩定現金流來源。[1][2][5]

**市場概況**
