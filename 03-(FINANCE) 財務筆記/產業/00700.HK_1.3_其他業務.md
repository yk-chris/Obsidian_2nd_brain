## 生產要素成本分析

### 智慧零售 & 地圖服務

**市場概況**

騰訊智慧零售為品牌商提供微信生態內的私域運營工具（小程序商城、會員體系、CRM），服務沃爾瑪、永輝、GAP 等超 500個零售品牌。中國智慧零售科技市場規模約 RMB 300億（2024），競爭對手包括阿里巴巴零售通及京東數科。騰訊地圖 API 服務主要面向企業開發者，年付費 API 調用量超 500億次，與高德（阿里）形成雙寡頭格局。
