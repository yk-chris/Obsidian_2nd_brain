# GOOGL — Google Cloud

## 業務部門概覽
Google Cloud 提供雲端基礎設施（IaaS）、平台服務（PaaS）、軟體即服務（SaaS），整合 AI/ML 解決方案如 Vertex AI、Gemini 模型。目標企業客戶與開發者，提供資料分析、大數據、AI 訓練等服務，是公司第二大成長引擎。（72字）[4][5][8]

## 第二部分：市場概況

全球雲端市場規模 8000 億美元，預計年成長 20%+，AI 算力需求為主要驅動。競爭格局為寡頭，AWS（31%）、Azure（20%）、Google Cloud（12%）三強鼎立。Google 憑藉 TPUs 硬體優勢、Gemini 第一方模型、開源工具（如 Kubernetes）建立差異化，積存訂單達 1550-2000 億美元。成長最快，連續多季盈利，利潤率持續提升。（128字）[1][4][5][7]

## 第四部分：投資回報分析

- **投入資本（Invested Capital）**：約 300 億美元，重資產數據中心。
- **ROIC / 部門資本回報率**：15%（快速改善中）。
- **FCF 貢獻**：轉正並成長，預計 2026 年貢獻 20%+ 公司 FCF。
- **成長驅動因素**：AI 算力需求、Gemini 生態、大企業遷雲。
- **競爭地位**：第三大，AI 領域最具潛力挑戰者。[1][4][5][7]
