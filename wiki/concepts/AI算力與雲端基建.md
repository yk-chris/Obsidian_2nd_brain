# AI 算力與雲端基建

## 核心命題
> AI 推理需求爆發 → 算力成稀缺資源 → 雲廠基建投資週期 → 2028 年起大規模兌現推理紅利

## 產業鏈層次

```
應用層（OpenAI / Claude / Gemini）
    ↓ 租用算力
雲端算力層（MSFT Azure / AWS / Oracle Cloud / Google Cloud）
    ↓ 採購 GPU & 伺服器
晶片層（NVDA GPU / AMD / 自研 ASIC）
    ↓ 需要先進製程
晶圓代工（TSMC / Samsung）
    ↓ 需要光刻設備
光刻設備（ASML EUV / DUV）
```

## 當前週期特徵（2025–2028）

### 供給端
- 先進節點與存儲供應緊張至少持續至 **2027 年**
- ASML 訂單排至 2027–2028 年
- 晶圓廠（TSMC、Samsung、Intel）均在大規模擴產
- High-NA EUV 預計 2027 出貨、2028 全面放量

### 需求端
- 主力驅動：AI 訓練 + AI 推理（推理需求增速快於訓練）
- 模型公司算力緊缺，開始「降智」現象（算力不足導致模型性能受限）
- 3–5 年內算力仍是最稀缺資源

### 雲廠節奏
| 階段 | 時間 | 特徵 |
|------|------|------|
| 建設期 | 2023–2027 | 大量資本開支，收入確認滯後 |
| 兌現期 | 2028+ | 推理訂單大規模交付，收入爆發 |

## 投資含義

### 確定性較高
- **ASML**：光刻設備獨家壟斷，訂單能見度高，但估值已高（PEG 1.67）
- **NVDA**：AI 算力直接受益，PEG 0.6（截至 2026-04）
- **TSMC**：AI 先進製程核心代工，PEG 0.8

### 潛在紅利（2027–2028）
- **MSFT / ORCL**：雲算力租賃，AI 推理收入 2028 年起大規模兌現；現技術支撐位 MSFT $393、ORCL $160

### 風險點
- MATCH 法案對 ASML 中國業務衝擊（~15% 總收入）
- OpenAI 等模型公司若失敗，間接影響雲算力需求
- 算力過剩風險（2028 年後）
